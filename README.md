<h1 align="center">Generate Short URL</h1>

## 🏗 Built With:
 - React.
 - HTML.
 - CSS.
 - JavaScript.
 - Bootstrap
 - Node Js.
 - Express Js.
 - MongoDB.


### Backend:
- Switch to Backend folder: 
```bash
cd backend
```
- Install required dependencies with: 
```bash
npm install
```
- Run with: 
```bash
nodemon index.js
```

### Frontend:
- Switch to Frontend folder: 
```bash
cd client
```
- Install required dependencies with: 
```bash
npm install
```
- Run with: 
```bash
npm start
```
